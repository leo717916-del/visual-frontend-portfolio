# AI 辅助视觉前端作品集主页

这是一个可以长期更新的作品集入口。以后对外展示时，优先发这个主页链接，而不是每次发单个作品链接。

## 使用方式

- 线上地址：https://visual-frontend-portfolio.vercel.app/
- GitHub 仓库：https://github.com/leo717916-del/visual-frontend-portfolio
- 页面结构：`index.html`
- 视觉样式：`styles.css`
- 作品数据：`projects.js`
- 页面交互：`script.js`

新增作品时，优先更新 `projects.js`，然后重新部署。客户访问同一个作品集链接时，就能看到最新版作品。

## 当前包含

- Forest Return：森林共生沉浸式落地页
- Stilllight Studio：摄影工作室预约落地页

## 对外用途

- 猪八戒服务介绍
- 闲鱼接单介绍
- 客户聊天中快速展示能力
- 个人主页/签名链接
- 后续作品集总入口
